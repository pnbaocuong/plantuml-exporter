## 🚀 **Why Use PlantUML Exporter?**

### ⚡ **Zero Setup Required**
```bash
# Install once
git clone https://github.com/pnbaocuong/plantuml-exporter.git
cd plantuml-exporter
make install

# Use anywhere
cd your-project-with-puml-files
plantuml-exporter

# Done! ✨
```

- ✅ **VSCode Compatible**: Same config as VSCode PlantUML extension
